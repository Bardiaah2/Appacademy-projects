Headings

Headings from 'h1' to 'h6' identify blocks and sections. Every page must have one h1.

Paragraphs

Paragraphs of texts go in 'p' tags

Lists

Unordered List

Unordered lists are marked with bullet points and go in the 'ul' tag. Each item list goes in a 'li' tag.

Type of lists:

  • Unordered lists ('ul')
  • Ordered lists ('ol')

Ordered List

Ordered lists are numbered and go in the 'ol' tag. Each item goes in a 'li' tag.

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the 'href' attribute. Go to the official site of the University of Oregon

Setting 'target="_blank"' will open the link in a new window. See MDN for more details.

Anchor tags can also create links to a new location on the same page. Instead of putting a URL goes in the "href" attribute, use the "id" of the element you want to navigate to. This allows you to Jump to the top of the page.

Text Emphasis

Italic

The 'em' element adds extra emphasis to the text within the opening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use the 'strong' element to emphasize text. This shows up as bold text in most browsers.
