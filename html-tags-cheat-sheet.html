HTML Tags Cheat Sheet

Headings

Headings from 'h1' to 'h6' identify blocks and sections. Every page must have one h1.

Paragraphs

Paragraphs of texts go in 'p' tags

Lists

Unordered List

Unordered lists are marked with bullet points and go in the 'ul' tag. Each item list goes in an 'li' tag.

Type of lists:

  • Unordered lists ('ul')
  • Ordered lists ('ol')

Ordered List

Ordered lists are numbered and go in the 'ol' tag. Each item goes in a 'li' tag.
